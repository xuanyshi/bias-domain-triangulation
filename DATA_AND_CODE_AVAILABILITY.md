# Data and Code Availability statement (for manuscript)

Suggested text for the manuscript:

---

**Data availability**

All data supporting the findings of this study are available in this article, its Supplementary Information and Source Data files, and in the public GitHub repository at https://github.com/xuanyshi/bias-domain-triangulation. No individual-level participant data were used; all extractions are from published study reports. The current Source Data include 39 adjusted estimates from 24 independent articles, per-record main-model covariate sets, bias-domain ratings, extraction-adjudication and inter-rater reliability results, the locked construct mapping, and LLM-generated candidate variables.

**Code availability**

All analysis code, including the R scripts for the REML random-effects meta-analysis with Knapp–Hartung inference, categorical moderator tests and sensitivity analyses, the Python scripts for the bias-domain rating engine and the LLM-based background DAG induction (Track B), and the exact prompts sent to the three large language models, are available at https://github.com/xuanyshi/bias-domain-triangulation. A step-by-step tutorial for applying the framework to other exposure–outcome questions is hosted at https://xuanyshi.github.io/bias-domain-triangulation/.
